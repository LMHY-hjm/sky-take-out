# 苍穹外卖 (Sky-Take-Out)
基于 Spring Boot 的餐饮外卖系统，用于学习和实践后端开发常用技术栈。
## 项目简介
本项目实现了外卖系统中的核心业务场景，包括用户端的菜品浏览、下单，以及管理端的菜品和套餐管理。并未完全根据原版视频填写，支付功能直接跳过，图片存储也由阿里云换成了本地存储
## 技术栈
- **后端**：Spring Boot、MyBatis、MySQL、Redis
- **接口文档**：Knife4j (Swagger)
- **工具**：Maven、Git、Postman
- ## 学习收获
通过这个项目，完整实践了从接口设计到数据库交互的后端开发流程，学习了如何使用 Redis 缓存热点数据，加深了对 Spring Boot 生态的理解。
同时也开始使用GitHub以及git进行代码管理
